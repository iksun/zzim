# 스프링 부트 프로젝트 압축 파일 전송 및 실행 가이드

1. **프로젝트 압축 파일 전송:**
   - 스프링 부트 프로젝트 폴더를 압축하여 파일을 받은 컴퓨터로 전송합니다.

2. **압축 파일 풀기:**
   - 명령어를 사용하여 압축 파일을 푸세요.
     ```bash
     unzip your_project.zip
     ```
     또는
     ```bash
     tar -xvf your_project.tar.gz
     ```

3. **프로젝트 폴더로 이동:**
   ```bash
   cd your_project
